# Safeguarding mallee birds — habitat suitability maps

This site presents interactive habitat suitability maps for the mallee bird community, linking expert-elicited State-and-Transition Models of vegetation condition to species- and community-level suitability scores.

Suitability is shown on a 0–7 Likert scale:

- 0 = species not expected to occur
- 7 = best available habitat
